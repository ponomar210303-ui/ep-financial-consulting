# Как добавить новую статью в блог

## Быстрый старт

1. Создай файл в папке `src/content/blog/` с расширением `.md`
2. Имя файла = slug (URL) статьи, например: `kak-platit-nalog-2026.md`
3. Заполни frontmatter и текст статьи
4. Сохрани — статья появится автоматически

---

## Структура файла

```markdown
---
title: "Заголовок статьи"
slug: "url-slug-stati"
date: "2026-04-07"
category: "Налоги"
excerpt: "Краткое описание статьи — отображается в карточке на главной и в списке блога (1-2 предложения)."
image: ""
readingTime: 5
---

Текст статьи в формате Markdown...
```

### Поля frontmatter

| Поле | Обязательно | Описание |
|---|---|---|
| `title` | да | Заголовок статьи |
| `slug` | да | URL (только латиница, цифры, дефис). Должен совпадать с именем файла. |
| `date` | да | Дата публикации в формате `YYYY-MM-DD` |
| `category` | да | Одна из категорий (см. ниже) |
| `excerpt` | да | Краткое описание для карточки |
| `image` | нет | Путь к изображению, например `/images/blog/my-image.jpg`. Оставь пустым если нет. |
| `readingTime` | да | Примерное время чтения в минутах (число) |

### Доступные категории

- `Налоги`
- `Živnosť`
- `Лайфхаки`
- `Бизнес`

## Пример новой статьи

Файл: `src/content/blog/kak-snizit-odvody-2026.md`

```markdown
---
title: "Как легально снизить odvody в 2026 году"
slug: "kak-snizit-odvody-2026"
date: "2026-04-10"
```

## Изображения для статей

1. Добавь изображение в папку `public/images/blog/`
2. В frontmatter укажи: `image: "/images/blog/имя-файла.jpg"`

Рекомендуемый размер: **1200×630px** (соотношение 16:9).

---

## Порядок статей

Статьи сортируются по полю `date` — от новых к старым. Самые новые появятся первыми в списке и на главной странице.

---

## После добавления

Статья появится автоматически:
- В разделе `/blog` — в общем списке
- На главной странице — в секции «Полезное» (показываются 3 последних)
- По прямой ссылке `/blog/твой-slug`

Перезапуск сервера или пересборка не нужны в dev-режиме — Vite подхватит новый файл после обновления страницы.
